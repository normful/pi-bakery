# pi-bakery

Pi extensions and maintained forks.

| Package                                                        | Description                                                                            |
| -------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| [`@normful/pi-show-files-read`](./packages/pi-show-files-read) | `/show-files-read` command — shows every file the agent has read this session          |
| [`@normful/pi-statusline`](./packages/pi-statusline)           | Info-rich TUI header/footer: context %, tokens, streaming CPS, cost, model, git branch |

```bash
pi install npm:@normful/pi-<name>
```

See each package's README for details.
